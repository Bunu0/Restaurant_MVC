Restaurant MVC Application
📝 Description

This is a Restaurant Management Web Application built using the MVC (Model-View-Controller) architecture.
It allows users to manage restaurant operations such as adding menu items, handling orders, and maintaining data efficiently.

🚀 Features
Add, update, delete menu items
Manage customer orders
View restaurant data in structured format
Clean MVC architecture for better scalability
🛠️ Tech Stack
Backend: Java, Spring Boot
Frontend: HTML, CSS
Database: MySQL
Build Tool: Maven
